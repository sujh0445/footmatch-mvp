# FootMatch

FootMatch는 신발 구매 시 사이즈 실패를 줄이기 위한 **프로토타입 의사결정 보조 서비스**입니다.

## 제품 핵심 로직 (현재 프로토타입)

- **실제 발길이(mm)** 를 사용자가 직접 입력합니다. (핵심 기준)
- 사진 분석은 길이를 정밀 측정하지 않고, **발 형태 경향(발볼/발가락/압박 가능성)** 을 참고용으로만 반영합니다.
- 추천 로직에서 **앞볼 압박 경험**과 **발볼 때문에 크게 사는 습관**을 우선 반영합니다.
- 결과는 정답 제시가 아니라, 구매 판단을 돕는 참고 정보입니다.

## 주요 화면

1. 랜딩 페이지
2. 발 정보 입력 (필수/선택 구분)
3. 발 사진 업로드 (선택)
4. 사이즈 제안 + 추천 이유 + 참고 프로필
5. 신발 검색
6. 신발 상세 + 나와 비슷한 발의 핏 리뷰
7. 핏 리뷰 작성

## 로컬 실행 방법

### 1) 요구사항

- Node.js 18.17+ (권장: Node.js 20 LTS)
- npm 9+

### 2) 프로젝트 폴더 이동

```bash
cd /workspace/footmatch-mvp
```

### 3) 의존성 설치

```bash
npm install
```

### 4) 개발 서버 실행

```bash
npm run dev
```

### 5) 브라우저 접속

- http://localhost:3000

## 스크립트

- `npm run dev` : 개발 서버
- `npm run build` : 프로덕션 빌드
- `npm run start` : 빌드 결과 실행
- `npm run lint` : ESLint 검사
